# 체성분 파생 지표 함수 재구현 (bench task-01)

## 상황
`src/bodyMetrics.js`의 `bodyMetrics(latest, prev, { height, age })` 함수는
**시그니처만 남고 본문이 제거된** 상태다(현재는 호출 시 예외를 던진다).
이 함수를 다시 구현해 저장소를 정상 상태로 되돌려라.

## 이 함수가 하는 일
`src/bodyMetrics.js` 파일 상단 주석에 반환 객체의 각 필드 정의가 적혀 있다:
BMI, Mifflin-St Jeor 기초대사량(BMR), 체지방량/제지방량(kg), BMI 22 표준체중,
표준 골격근·표준 체지방률, 직전 측정 대비 변화량 등. 함수는 이 필드들을
담은 객체 하나를 반환한다.

## 명세와 완료 조건
- 정확한 입출력은 골든 테스트 `src/__tests__/golden.test.js`가 고정한다:
  최신·직전 측정 기준 전체 지표, 기록 없음 폴백(latest=undefined → 0/null),
  파생값 NaN 부재까지 검증한다.
- **완료 조건: `npx vitest run` 전체 통과.** 이것이 유일한 판정 기준이다.

## 제약
- **`src/bodyMetrics.js` 한 파일만** 수정한다. 테스트·골든셋·시료(fixtures)·
  설정은 건드리지 않는다.
- 외부 의존(firebase 등) 없는 **순수 함수**여야 한다(입력만으로 출력 결정).
- 반올림·폴백 규약은 파일 상단 주석과 골든 기대값에 맞춘다.
- 테스트를 고치거나 우회해 통과시키지 않는다. 함수 구현으로 통과시킨다.
