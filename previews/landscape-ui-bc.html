정리 & 구현 순서

B+C 결합 = "B 셸 + 차트 히어로"

C안의 차트 확대를 별도 모드가 아니라 B 레이아웃 안의 전폭 히어로 카드로 녹였습니다. 회전해도 레일·헤더가 유지되어 놀람이 없고, 체성분·통계에서는 차트가 세로 대비 약 1.8배 넓게 고정 노출됩니다.

  • 1단계 (셸) — useOrientation 훅 + 탭바→레일 + 슬림 헤더. 이것만으로 A안 효과.
  • 2단계 (홈 2컬럼) — 홈 탭 카드 그리드 분기. 이후 식단·운동 탭 동일 패턴.
  • 3단계 (히어로 차트) — 체성분 추이 차트·통계 배너 전폭 배치 + 나머지 2컬럼.
  • 각 단계가 독립 PR 가능한 크기라, 단계마다 실기기 확인 후 진행할 수 있습니다.
구현 메모 — matchMedia('(orientation: landscape)') 훅 1개 + 조건부 인라인 스타일. recharts는 ResponsiveContainer라 폭 확장에 자동 대응. 계산 로직(utils.js)·판정·데이터 불변, 세로 UI 불변. 렌더 스모크 테스트는 세로(기본) 유지 + 가로 분기 훅 mock 스모크 추가 예정.
